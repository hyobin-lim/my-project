# 시스템 구조도 (System Architecture)
*최종 업데이트: 2026-02-26 17:50:00*

## 1. 프론트엔드 계층 구조 (Frontend Layer)
- **컴포넌트 설계**: Atomic Design 원칙에 기반한 세분화.
    - `Header/`: Logo, GNB, SearchBox, MegaMenu (상태 및 데이터 분리)
    - `CultureCard/`: 고밀도 정보 그리드 레이아웃 (3:4 비율 지향)
    - `DetailModal/`: 상세 정보 및 외부 링크 연결
- **상태 관리**: React `useState`를 활용한 카테고리 및 필터링 제어.
- **스타일링**: 컴포넌트별 CSS 모듈화 (Scoped CSS).

## 2. 데이터 분류 체계 (Data Taxonomy)

### **[중분류 (L2): 🎭 공연 세부 분류 (데이터 밀도 중심 6대 그룹)]**
| 세부 분류 (L2) | 세부 항목 및 범위 (L3) | 내비게이터 팁 |
| :--- | :--- | :--- |
| **연극/뮤지컬** | 초대권 응모, 기업/통신사 혜택, 선착순 예약, 학생/아마추어 공연 | **응모형** 고가치 데이터 |
| **클래식/오페라/무용** | 독주/협주회, 교향악, 합창/성악, 오페라, 발레, 현대무용, 한국무용 | 장르별 **전문성** 확보 |
| **국악/전통예술** | 판소리/민요, 사물놀이/풍물, 탈춤/마당놀이, 전통 의례/제례악 | 우리 문화 **독자 영역** |
| **대중음악/버스킹** | 인디/록, 재즈, K-POP/아이돌, 트로트/성인가요, 포크/발라드, 음악 축제, 거리공연 | **트렌드** 및 타겟 최적화 |
| **어린이/가족** | 인형극, 마술쇼, 어린이 뮤지컬, 구연동화, 가족 체험/워크숍 | **육아** 타겟 최적화 |
| **무장애/시니어/이벤트** | 배리어프리 공연, 시니어 우대공연, 갈라쇼, 북 콘서트, 복합예술 | **배려/이색** 타겟 통합 |

### **[중분류 (L2): 🎬 영화 세부 분류 (데이터 논리 중심 7대 그룹)]**
| 세부 분류 (L2) | 세부 항목 및 범위 (L3) | 내비게이터 팁 |
| :--- | :--- | :--- |
| **시사회/기업혜택** | 시사회/초대 응모, 통신사/카드사 우대, 0원 딜/선착순 할인권 | **파격적** 비용 절감 |
| **공공/도서관 상영** | 공공도서관, 구청/시민강당, 미디어센터, 박물관/미술관, 자치회관 | **생활 밀착형** 정보 |
| **야외/축제/자동차** | 한강/공원 야외상영, 자동차 극장, 지역 축제 연계 상영, 찾아가는 영화관 | **낭만/이벤트** 상영 |
| **영화제/특별상영** | 국내외 국제영화제, 시니어/장애인 영화제, 어린이/청소년/애니 영화제, 독립/예술 영화제 | **[집중]** 축제성 영화 정보 |
| **독립/예술/시네마테크** | 독립/예술영화 상영, 시네마테크 기획전, 영상자료원(KOFA), 예술영화 전용관 | **[상시]** 전문 상영관 경로 |
| **무장애/시니어 우대** | 배리어프리(가치봄) 상영, 시니어 우대 상영, 시니어 전용관, 화면해설/수어통역 | **상시 시설 및 서비스** |
| **어린이/가족/애니** | 어린이/가족 상설상영, 애니메이션 상영관, 키즈 전용관 체험, 만화박물관 기획상영 | **아이들을 위한** 상설 정보 |
| **무장애/시니어 우대** | 배리어프리(가치봄) 상영, 시니어 우대 상영, 시니어 전용관, 화면해설/수어통역 | **특수 타겟** 접근성 |
| **어린이/가족/애니** | 어린이/가족 영화, 애니메이션 상영, 키즈 전용관 체험, 만화박물관 기획상영 | **가족 단위** 주말 정보 |

| 세부 분류 (L2) | 세부 항목 및 범위 (L3) | 내비게이터 팁 |

| **어린이/가족** | 어린이 미술교실, 가족 체험 워크숍, 키즈 전용관, 그림책 도서관 | **육아/체험** 실익 정보 |
| **무장애/시니어 우대** | 점자/촉각 전시, 시니어 우대 관람, 시니어 예술체험, 수어 해설 전시 | **포용적** 관람 환경 |

## 3. 데이터 흐름 (Data Flow)
1. **Collector**: 사이트별 맞춤형 스크래퍼가 원천 데이터 수집.
2. **Refined Store**: AI 에이전트(Organizer)가 수집된 데이터를 L2/L3 카테고리로 분류 및 정제.
3. **Web Interface**: 정제된 데이터를 사용자에게 큐레이션 대시보드 형태로 제공.
