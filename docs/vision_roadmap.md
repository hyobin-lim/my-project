# 🏛️ 비전 및 로드맵 (Vision & Roadmap)
*최종 업데이트: 2026-02-24 17:56:57*

## 1. 프로젝트 개요
- **목표**: 문화생활 통합 플랫폼 개발 (웹 & 앱) 및 후원 모델 구축.
- **핵심 가치**: 흩어진 무료 문화 정보를 한곳에 모아 제공하는 '원스톱 서비스'.
- **파트너십**: AI와 파트너(사용자)가 동등한 입장에서 협력하며 함께 성장하는 프로젝트.

## 2. 기술 스택 (Tech Stack)
- **Frontend**: React (Vite 6), TypeScript, Vanilla CSS.
- **Backend/Collector**: Python (requests, BeautifulSoup/Selenium).
- **Data**: JSON 기반 파일 스토리지 (추후 DB 확장 고려).
- **Environment**: Node.js v24.13.0, Python 3.14.2.

## 3. 장기 로드맵
- **Phase 1 (기초)**: 웹 UI 뼈대 및 데이터 수집 로봇 기초 구축. (진행 중)
- **Phase 2 (연동)**: 수집된 실제 데이터를 웹 화면에 동적으로 렌더링.
- **Phase 3 (고도화)**: 필터링, 검색, 상세 페이지 및 사용자 편의 기능 강화.
- **Phase 4 (배포)**: PWA(Progressive Web App) 적용 및 실제 서비스 런칭.
