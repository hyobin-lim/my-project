### **1 기술 스택**
- **Frontend**: React (Vite 6), TypeScript (TSX), Vanilla CSS.
- **Backend/Collector**: Python (requests, BeautifulSoup, Selenium).
- **Data Storage**: JSON 기반 파일 스토리지 (`store/raw`, `store/refined`).


### **2 구현 원칙**
- **철저한 모듈화**: 하나의 파일에 많은 로직을 담지 않고, 기능별로 컴포넌트와 훅(Hook)을 분리한다.
- **Atomic Design**: 컴포넌트를 아토믹 단위로 세분화하여 재사용성을 높인다.
